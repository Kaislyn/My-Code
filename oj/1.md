
# P2022 人工智能的相亲问题

## 题目描述
人工智能现在也有性别了，我们不妨用 0/1 表示，因此这些人工智能也有了相亲的需求。如何评价两个人工智能是否足够匹配呢？把 LevOJ 上的题做一遍，如果两个人工智能的 AC 题数越相近就越适合被匹配到一起，将差值的绝对值设为 D，并且基于此设定阈值 D<sub>s</sub>，不满足 D <= D<sub>s</sub> 的人工智能将不能被匹配起来。

在这个诡异的平行宇宙里，你需要给 M 个性别为 0 的人工智能和 N 个性别为 1 的人工智能进行配对，请问阈值 D<sub>s</sub> 最少为多少时，所有人工智能都可以找到至少一个匹配对象（每个人工智能都可以被多次用于匹配）？

## 输入描述
第一行包括两个正整数 M (1 <= M <= 10^5) 和 N (1 <= N <= 10^5)，用空格隔开，分别表示性别为 0 与性别为 1 的人工智能的个数。

第二行有 M 个正整数，均不超过 10^6，分别表示每个性别为 0 的人工智能的 AC 题目数。

第三行有 N 个正整数，均不超过 10^6，分别表示每个性别为 1 的人工智能的 AC 题目数。

## 输出描述
输出阈值 D<sub>s</sub> 的最小值，使得所有人工智能都可以找到至少一个匹配对象。

## 样例输入 1
```
6 5
1 6 3 5 4 2
9 8 3 11 6
```

## 样例输出 1
```
5
```

## 样例输入 2
```
10 5
42 68 35 1 70 25 79 59 63 65
6 46 82 28 62
```

## 样例输出 2
```
8
```
